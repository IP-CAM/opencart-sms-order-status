Пришлось сделать разные версии под разные сборки Opencart:
1. Версия для чистого Opencart - функционала СМС в нем нет изначально.
2. Версия для ocStrore - изначально есть только функционал смс оповещений администратора
3. Версия для MaxyStore - наполвину "украденный" функциоанл смс из ocStore - настройки в админке есть, а кода посылки СМС нет :-)

В связи с такими разлиями, привести модуль к одной универсальной версии не получилось, поэтому пришлось сделать 3 разных копии.

Для тестов удобно использовать:
1. smstest.xml - добавляет кнопку посылки тестовой СМС в админке
2. testsms.php - специальный смс шлюз, который пишет все посылаемые смс в лог файл (system/logs/error.txt) без реальной посылки абоненту.

Каждый СМС шлюз лежит в отдельной папке.
