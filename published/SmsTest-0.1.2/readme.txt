Отправка тестовой СМС
Автор: Gennady Telegin <support@itxd.ru>

Добавляет в админку кнопку для отправки тестовой СМС через настроенный по умолчанию шлюз.

Для установки скопируйте содержимое папки "upload" на ваш сервер. Требуется установленный VQMod.

Все вопросы и предложения пишите на support@itxd.ru
