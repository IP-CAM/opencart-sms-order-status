Модуль для работы с СМС шлюзом MainSms.Ru
Автор: Gennady Telegin <support@itxd.ru>

Установка и настройка:
1. Скопируйте содержимое папки upload в корень вашего сервера.
2. В личном кабинете mainsms.ru перейдите на вкладку API и добавьте своей проект
3. Дождитесь окончания модерации
3. Перейдите в настройки магазина, вклада SMS
4. Задайте следующие поля:
  Шлюз - mainsms
  Логин - имя проекта
  Пароль - API key
  Имя отправителя - настроенное в mainsms
  
В случае вопрос пишите на support@itxd.ru
