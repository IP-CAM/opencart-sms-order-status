Модуль СМС-шлюз bytehand.com
Автор: Gennady Telegin, support@itxd.ru

Для установки скопируйте содержимое каталога upload на Ваш сервер.

Настройка:
В настройках магазина на вкладке СМС, заполните поля следующим образом:
1) Сервис отправки SMS - bytehand
2) Логин на SMS шлюз - ваш ID (4-х цифровой код, написан в личном кабинете bytehand, Аккаунт->Настройки)
3) Пароль на SMS шлюз - ваш Ключ (написан в личном кабинете bytehand, Аккаунт->Настройки)
4) Псеводним отправителя - любая строка, зарегистрированная в личном кабинете bytehand, Рассылки->Подписи отправителя
